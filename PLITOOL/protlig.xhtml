PROT-LIG

Prot-Lig is developed for protein-ligand interaction data collection. There are many databases such as ChEMBL, PDB, and PubChem; however, ChEMBL is most dominant and preferrable database. Thus, Prot-Lig extracts ligand data from ChEMBL and protein data from UniProt. The main benefits of this tool is to collect protein-ligand data programmatically.

In this tool, first the protein data are collected from UniProt, then ligand data are extracted from ChEMBL database. The output of the tool comprises protein ID, ligand ID, and SMILES.

Users can specify the search protein name, organism number and even the threshold. Protein name might be a UniProt ID (or IDs), and proteins' keywords. For organism number,user should decide for the organism for which the data are collected. The organism numbers are also provided in our webpage. For instance, 9606 is used for human. For thresholding, default threshold is mostly preferred, which is 1000 nM.Here, threshold value is used for ligands' standard values. Ligands having equal or smaller standard values than the threshold are collected. The threshold value (1000nM) is recommended by the founder of ChEMBL database. However, users might also calculate the trheshold by selecting the other option. For that, the program calculates the mean of the standard values, then these mean value is used as the threshold.
