PLITOOL Description

PLITOOL

What is PLITOOL?

PLITOOL (Protein-Ligand Interaction TOOL) provides a multi-purpose platform to obtain and analyze protein-ligand interactions. It combines three different functionalities:

  • Extraction of protein-ligand interactions from CHEMBL databases.
  • Visualization of protein-protein interactions on different network models based on their ligands.
  • Computation of compound similarity with three different methods.

What is Protein-Protein Network?

A protein-protein network is a representation where proteins are shown as the nodes of a network and connected through a feature. Protein-protein networks enables better investigation of protein interactions with the help of network analysis metrics. Clustering is one of the most powerful approaches in network models to investigate densely connected sub-groups. PLITOOL provides a ligand-centric network model [cite] for a set of proteins which is previously proposed by our team.

User Inputs

PLITOOL requires from user to decide (1) protein name/protein family name/UniProt ID/Gene Ontology ID and (2) organism name.
